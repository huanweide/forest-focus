## ✨ 功能

- 🍅 **番茄钟** 20/25/30/45/60分钟 · 紫色渐变计时环
- 🌳 **种树系统** 每完成专注解锁阿梓新衣装 · 共14级
- 👗 **衣柜画廊** 查看所有阿梓形象+解锁进度
- ✅ **习惯打卡** 三级难度 · 连续天数 · 健康值 · 月热力图
- 🎯 **目标追踪** 番茄数进度 · 手动增量
- 📊 **数据仪表盘** 饼图 · 连续天数 · 今日/本周/本月/本年切换
- 🏆 **17项成就** 进度条 · 🥉🥈🥇👑四级稀有度
- 🔒 **专注锁定** 切走页面触发遮罩
- 🌙 **暗色模式** 一键切换
- 📱 **PWA** 添加到主屏幕 · 离线可用

## 📱 安装到手机

1. 用浏览器打开 [应用链接](https://huanweide.github.io/forest-focus/)
2. iPhone: 点分享 → 添加到主屏幕
3. Android: 点菜单 → 添加到主屏幕
4. 主屏出现阿梓图标 → 点击即全屏运行
